# Medtech Hackathon 2025

This repository contains prototypes for a Streamlit-based medical application, machine-learning utilities, and supporting infrastructure.

## Getting Started

1. Install dependencies with `uv sync --group ml-torch --group llm-ollama`.
2. Launch the Streamlit app via `uv run streamlit run app.py` (replace with the actual entry point).
3. See `databases/database.py` for SQLite schema helpers.
